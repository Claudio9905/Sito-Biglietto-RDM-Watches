# Obiettivo
Una web application, responsive fino agli schermi dei tablet, che rappresenta un biglietto da visita digitale, tramite scansionamento del codice QR code, per l'azienda RDM Watches,
in modo da avere sempre, a portata di mano, il contatto con l'azienda.

# Struttura
L'app presente principalmemnte i contatti, tra link del sito principale dell'azienda, link dei social, link di Google Maps per la posizione GPS
